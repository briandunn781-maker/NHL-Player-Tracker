Dunn_NHLTop_100 Database Application: 

This application is a NHL database application with real NHL data that you can extract built for NHL fans! 

Ensure mysql connector is running locally before procedding with run of GUI.py

Versions: 
Python: 3.13.5
mysql-connector0-python: 9.4.0


#Buttons: 
Player team finder: Look up NHL teams and find players attached to that team. 
Player Add: Add a new NHL player to this database to any team
Player Mover: Move a player from one team to another. 
Team USA: Find and build your own Team USA roster. 
Player Delete: Delete a player from your databse. 

#Advanced Feature: 

-Player Guessing game: This game is desinged to generate a random player player profile, you will make your guess and the result will appear in the text box below. If you would like to keep playing, please hit the 'redraw' button to proceed. 

#Installation Info: 
- All credentials will be set an instantiated inside the GUI. Any credentials that are incorrect for database connection will not work with this GUI applicaiton. 
-Follow all prompts inside GUI view layer for direction on how to use this application, you will be given prompts and response boxes to properly input data. 


Launch Gui.py root folder //Applicaton Layer/Gui.py

When you launch your Gui.py file, all necesarry files will be included in the zip file.

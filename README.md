# NHL-Player-Tracker

## Project Scope

This project is a simple NHL player tracking system that provides a local database (SQLite) of player stats and lets users query, add, and update player information via a lightweight Python GUI application.

The core components include:
- **Data Access Layer (DAL):** Handles SQLite database operations.
- **Business Logic Layer (BLL):** Encapsulates application rules and data transformations.
- **GUI:** A user interface for searching and managing player records.
